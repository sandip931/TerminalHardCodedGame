# 🎮 Console Games Architecture Blueprint

## Project: Tic-Tac-Toe Production Implementation

This repository will contain a clean, modular C++ implementation of a classic Tic-Tac-Toe game, designed with **SOLID principles** and **Separation of Concerns** in mind.

---

## 📁 Project Structure

```text
tictactoe/
├── 📄 Board.cpp          # Game board logic implementation
├── 📄 Board.h            # Game board declarations
├── 📄 Game.cpp           # Game flow control implementation
├── 📄 Game.h             # Game flow declarations
├── 📄 Player.cpp         # Player abstraction implementation
├── 📄 Player.h           # Player abstraction declarations
├── 📄 HumanPlayer.h/cpp  # Human input handler
├── 📄 AIPlayer.h/cpp     # Computer opponent (Minimax)
├── 📄 main.cpp           # Application entry point
└── 📄 Makefile           # Build automation script
